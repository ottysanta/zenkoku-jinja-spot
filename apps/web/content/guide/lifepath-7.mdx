## ライフパス7とは何か——数字の中に宿る魂の地図

### 計算方法

ライフパスナンバーは、生年月日のすべての数字を足し合わせ、最終的に1桁（またはマスターナンバー11・22・33）になるまで繰り返し還元して求めます。

たとえば、1987年8月23日生まれの場合：

```
1 + 9 + 8 + 7 + 8 + 2 + 3 = 38
3 + 8 = 11 → マスターナンバーのため11のまま
```

最終的に「7」になった方がライフパス7です。

自分の数字が分からない方は、診断ページで生年月日を入力するだけで自動計算されます。

### ライフパス7の本質：見えない世界を知りたい魂

数秘術において「7」は**霊的探求・内省・真理・孤独な完成**を象徴する数字です。

1〜9の数字の中で最も「内側に向かう」エネルギーを持ち、物質的な成功や外側の承認よりも、**目には見えない本質的な真実を知ること**に深い喜びを感じます。

一般的によく言われるライフパス7の特徴を列挙してみます：

- 一人の時間が必要で、それが深い充電になる
- 表面的な会話より、深い対話を好む
- 直感が鋭く、人の嘘や建前をすぐに感じ取る
- 哲学・心理学・スピリチュアル・宗教・科学の本質に惹かれる
- 完璧主義で、中途半端なものに強い違和感を覚える
- 「本当のことを言っているのはこの人だけ」という場所や人に出会うと、深く傾倒する

<KeyPoint title="ライフパス7が抱えやすい葛藤">
「もっと社交的にならなきゃ」「なぜ一人が好きなんだろう、変なのかな」と自分を責めるライフパス7の方は多いです。でも、あなたの内向性は欠点ではなく、深い洞察と真理への感受性の源泉です。
</KeyPoint>
